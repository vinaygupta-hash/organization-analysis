# Global Organization Data Analysis

## Project Overview
This project analyzes 1,000 organizations across countries and industries
using SQL for data analysis and Tableau for visualization.

## Dataset
- Source: Public organization dataset
- Records: 1,000 organizations
- Attributes:
  - Organization ID
  - Name
  - Country
  - Industry
  - Year Founded
  - Number of Employees
  - Revenue Category

## Tools Used
- MySQL (Data querying)
- Tableau (Data visualization)
- GitHub (Project documentation)

## Key Analysis Performed
- Country-wise organization distribution
- Industry-wise organization count
- Revenue contribution by country and industry
- Organization size distribution
- Top organizations by revenue

## Files in Repository
- data/organizations-1000.csv → Raw dataset
- sql/organizations_analysis.sql → SQL queries used
- tableau/organization_analysis.twbx → Tableau dashboard

## Visualizations
Interactive dashboards created using Tableau to present insights clearly.

## Outcome
This project demonstrates practical SQL querying, analytical thinking,
and business-focused data visualization skills.
